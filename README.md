# vvf-3-livello

Progetto web semplice per la consultazione delle domande VVF di 3 livello.

## Contenuto

- `index.html` → pagina principale.
- `manifest.json` → configurazione PWA.
- `icons/` → icone e immagine grafica dell'app.

## Note

Questa versione è pensata per funzionare online tramite GitHub Pages.

Non include:
- service worker
- cache offline
- logica di salvataggio locale
